This repository contains the tasks completed as part of the CodeAlpha AI & Machine Learning Internship.
 Task 1: Student Score Prediction
- Built using Linear Regression
- Predicts student marks based on study hours
- Visualized results using matplotlib
  
 Task 2: Salary Prediction
- Built using Linear Regression
- Predicts salary based on years of experience

Task 3: Customer Purchase Prediction
- Built using Decision Tree Classifier
- Predicts whether a customer will buy a product based on age

Technologies Used
- Python
- NumPy
- Scikit-learn
- Matplotlib

Objective
To understand and implement basic Machine Learning algorithms including regression and classification.
